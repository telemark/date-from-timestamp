# date-from-timestamp
Returns date on dd.mm.yyyy format from timestamp

## Installation

```sh
$ npm i date-from-timestamp
```

## Usage
```javascript
'use strict'

const dateFromTimestamp

console.log(dateFromTimestamp('1467712800000')) // => 05.07.2016

```
